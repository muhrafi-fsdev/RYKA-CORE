# RYKA CORE 4.4 Personal Access & Partner Communication

Developer: **Muhammad Rafi Priyo**

Versi 4.4 adalah compatibility upgrade dari RYKA CORE 4.3. Seluruh fitur kamera, MediaPipe, orb, Blur/Mosaic/Flip, dynamic gesture, action mapper, secure desktop bridge, HMAC signing, replay protection, rate limiting, dan security workflows tetap dipertahankan.

## Fitur baru

- RYKA Access full-screen suite.
- Gesture-to-Text dengan profile kalimat personal.
- Quick Phrase Board multi-kategori.
- Text-to-Speech lokal melalui browser/sistem.
- Live Caption berbasis Web Speech Recognition jika didukung.
- Conversation Mode dua arah.
- Visual Sound Alert berdasarkan level suara.
- Emergency Communication Mode.
- Favorites dan communication history lokal.
- Import/export profil gestur JSON.
- Teks besar, kontras tinggi, reduced motion, dan kontrol aksesibel.
- Launcher `START_ACCESS.bat`.
- Query startup `?mode=access`.
- Dokumentasi `ACCESSIBILITY.md`.

## Catatan batasan

- Tidak terhubung ke NusaMind AI.
- Belum menerjemahkan BISINDO atau SIBI.
- Live Caption dan TTS bergantung pada kemampuan browser dan sistem operasi.
- Visual Sound Alert belum mengklasifikasikan jenis suara.
- Bukan perangkat medis.
