# Resource References

RYKA CORE remains an original codebase. The following public projects were used as technical references for architecture, security controls, and interaction patterns:

## Vision and interaction

- Google AI Edge MediaPipe — hand landmarks and gesture recognition architecture
- casiez/OneEuroFilter — adaptive low-pass filtering for low-jitter input
- Viral-Doshi/Gesture-Controlled-Virtual-Mouse — gesture-to-desktop interaction ideas
- andypotato/fingerpose — rule-based custom gesture description ideas
- pmndrs/react-three-fiber and pmndrs/postprocessing — modular Three.js HUD direction

## Native and voice direction

- tauri-apps/tauri — future native desktop packaging
- tauri-apps/plugins-workspace Stronghold — future encrypted secret storage
- enigo-rs/enigo — future native mouse and keyboard input
- dscripka/openWakeWord and ggml-org/whisper.cpp — future offline voice assistant

## Cybersecurity references

- OWASP/ASVS — application security verification requirements
- OWASP/CheatSheetSeries — secure design and implementation guidance
- OWASP/threat-dragon — threat-modeling methodology
- google/osv-scanner — dependency vulnerability scanning
- github/codeql-action — semantic security analysis
- actions/dependency-review-action — vulnerable dependency review
- semgrep/semgrep — static analysis and secure coding rules
- gitleaks/gitleaks — secret detection concepts
- ossf/scorecard — open-source supply-chain security posture

No Marvel assets, names, audio, or copyrighted UI resources are bundled with RYKA CORE.
